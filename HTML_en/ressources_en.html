The project's resources

  • Unity

    This is the game engine that we mainly used to make our game. It can concentrate all the elements of a game, like assets and scripts.

  • Visual Studio

    IDE for C# development. It works extremely well in conjunction with Unity and allows you to code scripts.

  • Photon Unity

    Extension for Unity allowing us to make the multiplayer part of our game.

  • Visual Studio Code

    Code editor that allows us to code the website.

  • Clip Studio Paint

    Drawing software that is used to make the different image assets of the game.

  • GitHub Pages

    Software that allows us to host the website.

  • Git

    Code version management software. It facilitates the task of sharing code within the group by offering advanced and intelligent synchronization features.

  • Audacity

    Digital sound recording software. We used it to record the sound effects that we use in the game.
